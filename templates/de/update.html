Was ist Evolution?

DASH Evolution wird die Verwendung von Kryptowährungen so einfach machen wie nie zuvor und dafür sorgen, dass die vergleichsweise komplizierten Transaktionen von Kryptowährungen der Vergangenheit angehören. DASH-Transaktionen über Evolution werden so einfach wie die Verwendung von Paypal, aber mit einem entscheidenden Vorteil: DASH Evolution wird komplett dezentral und nicht gewinnorientiert arbeitet.

Sentinel

Ein mächtiger Werkzeugkasten für DASH. DASHs neue Sentinel Technologie ist der Motor unter der Haube von DASH Evolution. Sentinel ist leistungsstark, flexibel und anpassungsfähig, was die Updates für das Netzwerk extrem vereinfachen und eine Vielzahl neuer Anwendungen ermöglichen wird. So können mit Sentinel beispielsweise vollkommen digitale Unternehmen gegründet werden.

EVOLUTION FORSCHUNG

Im Januar 2016 veröffentlichten wir einen frühen Evolution Prototyp auf Basis von Electrum. Seit damals haben wir eine umfassende Dokumentation erstellt, unsere Ziele vollständig umrissen und mit der Umsetzung begonnen. Zwar war der Prototyp mit so grundlegenden Funktionen wie dem Zahlungsverkehr zwischen Freunden mit Benutzernamen über einen Dev-Server recht einfach, doch hat er bewiesen, dass das erdachte Prinzip funktioniert. Unser damaliges Ziel war es einen Prototypen für alle Schlüsseltypen zu designen, der die genannte Funktion auf reine P2P-Weise ermöglichen sollte, wobei die Daten auf der DASH-Blockchain gespeichert werden sollten. Dies sollte auf jedem Gerät mit integrierter SPV-basierter Sicherheit funktionieren, das keine zentralisierten Vermittlungsdienste benötigt und mit dem gebräuchlichsten und Zensur resistenten Protokoll liefen - HTTP (S)

Launch Date

Das Starten einer neuen Software-Version im Netzwerk erfordert, dass alle Budgetvorschläge an das System zurückgesendet werden. Durch das Timing der Superblocks, haben wir Anfang des Monat ein kurzes Fenster um das Update zu launchen. Um sicherzustellen, dass wir die stabilste Version der Software in Betrieb nehmen, ohne das Budget-System zu stören, müssen wir bis Anfang Februar 12.1 starten. Launch Date: 5. Februar 2017
